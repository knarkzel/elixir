# Elixir

Elixir is a reddit-like web application with the goals of being tiny, performant and easy to set up.

## Deploy

Rust is required. [rustup](https://rustup.rs/) is recommended.

```bash
git clone https://github.com/knarkzel/elixir
cd elixir/
cargo run
```
